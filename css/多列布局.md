# 多列布局

```css
.container {
  height: 800px;
  background-color: red;
  /* 列数 */
  column-count: 5;
  /* 间隔 */
  column-gap: 50px;

  /* 
    列边框
    column-rule-width: initial;
    column-rule-style: dashed;
    column-rule-color: red;
    的复合属性
  */
  column-rule: 1px solid red;
  /* 列内容填充模式
    当内容无法填充满时，auto默认填满一列后再填充下一列，balance则平均分配每一列的数据
  */
  column-fill: balance;

  /* 列宽 */
  /* 自动重新计算列数，使容器脱离column-count控制 */
  column-width: 500px;
}

.container>h1 {
  /* 避免元素在容器内中断 */
  /* break-inside: avoid; */

  /* 子元素垮列 */
  column-span: all;
  text-align: center;
}
```
